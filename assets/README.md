# Assets

This folder contains images, diagrams, icons, and other visual assets used across the KnowGrow Roadmaps repository.

## 📁 Suggested Asset Types

- `banner.png` — Main repository banner image
- `roadmap-python.png` — Visual Python roadmap diagram
- `roadmap-webdev.png` — Visual Web Development roadmap diagram
- `roadmap-aiml.png` — Visual AI/ML roadmap diagram
- `roadmap-systemdesign.png` — Visual System Design roadmap diagram
- `logo.png` — KnowGrow Roadmaps logo

## 🤝 Contributing Assets

When adding assets:
- Use **PNG** for diagrams and screenshots
- Use **SVG** for icons and logos (preferred for scalability)
- Keep file sizes reasonable (compress images before uploading)
- Use descriptive, lowercase, hyphen-separated filenames
