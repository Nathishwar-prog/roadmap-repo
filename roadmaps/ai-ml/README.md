# 🤖 AI / Machine Learning Roadmap

> **Goal:** Understand the fundamentals of AI and Machine Learning, build real models, and eventually deploy intelligent applications.

---

## 📌 Overview

| Stage | Topics | Estimated Time |
|---|---|---|
| 🟢 Beginner | Python, Math, Intro to ML | 6–8 weeks |
| 🟡 Intermediate | ML Algorithms, Deep Learning, NLP | 10–14 weeks |
| 🔴 Advanced | MLOps, LLMs, Research Papers, Production AI | 12–16 weeks |

---

## 🟢 Stage 1 — Beginner

**Goal:** Build the mathematical and programming foundation for ML.

### 🐍 Python for Data Science
- ✅ Python basics (see [Python Roadmap](../python/README.md))
- ✅ NumPy — arrays, matrix operations, broadcasting
- ✅ Pandas — DataFrames, data cleaning, manipulation
- ✅ Matplotlib & Seaborn — data visualization

### 📐 Math Foundations
- ✅ **Linear Algebra**: vectors, matrices, dot product, eigenvalues
- ✅ **Statistics**: mean, median, standard deviation, probability
- ✅ **Calculus**: derivatives, chain rule, gradient descent concept
- ✅ **Probability**: Bayes' theorem, distributions

### 🤖 Introduction to Machine Learning
- ✅ What is ML? Supervised vs Unsupervised vs Reinforcement Learning
- ✅ Key terminology: features, labels, training/testing, overfitting
- ✅ Your first model with Scikit-learn
- ✅ Train/test split, cross-validation

### 🛠️ Beginner Projects:
- 🏡 House price prediction (linear regression)
- 📧 Email spam classifier
- 🌺 Iris flower classification
- 🎬 Movie recommendation (basic)

---

## 🟡 Stage 2 — Intermediate

**Goal:** Master core ML algorithms and start using neural networks.

### 🧠 Core Machine Learning Algorithms
- ✅ Linear & Logistic Regression
- ✅ Decision Trees & Random Forests
- ✅ Support Vector Machines (SVM)
- ✅ K-Nearest Neighbors (KNN)
- ✅ K-Means Clustering
- ✅ Gradient Boosting (XGBoost, LightGBM)
- ✅ Model evaluation: confusion matrix, ROC-AUC, F1-score

### 🧬 Deep Learning Foundations
- ✅ What is a neural network? Neurons, layers, activation functions
- ✅ Backpropagation and gradient descent
- ✅ **TensorFlow** or **PyTorch** — pick one and stick with it
- ✅ Convolutional Neural Networks (CNNs) for images
- ✅ Recurrent Neural Networks (RNNs) for sequences
- ✅ Regularization: Dropout, Batch Normalization

### 💬 Natural Language Processing (NLP)
- ✅ Text preprocessing: tokenization, stemming, stopwords
- ✅ TF-IDF and word embeddings (Word2Vec, GloVe)
- ✅ Transformers and attention mechanism (conceptual)
- ✅ Hugging Face `transformers` library basics

### 🛠️ Intermediate Projects:
- 😊 Sentiment analysis on Twitter data
- 🖼️ Image classifier (cats vs dogs with CNN)
- 📰 News article topic classification
- 🤖 Simple chatbot with intent detection

---

## 🔴 Stage 3 — Advanced

**Goal:** Build and deploy production-grade AI systems.

### 🚀 Large Language Models (LLMs)
- ✅ Understanding transformer architecture (BERT, GPT)
- ✅ Fine-tuning pre-trained models
- ✅ Prompt engineering
- ✅ LangChain for building LLM applications
- ✅ Retrieval-Augmented Generation (RAG)
- ✅ OpenAI API, Anthropic Claude, Hugging Face models

### 🏭 MLOps (Machine Learning Operations)
- ✅ Model versioning with MLflow or Weights & Biases
- ✅ Building ML pipelines
- ✅ Deploying models as APIs (FastAPI + Docker)
- ✅ Model monitoring and drift detection
- ✅ Cloud ML platforms: AWS SageMaker, Google Vertex AI, Azure ML

### 📊 Advanced Topics
- ✅ Reinforcement Learning fundamentals
- ✅ Generative AI: GANs, VAEs, Diffusion Models
- ✅ Computer Vision: YOLO, object detection, segmentation
- ✅ Reading and implementing research papers

### 🛠️ Advanced Projects:
- 🤖 RAG-based document Q&A chatbot
- 🖼️ Image generation with Stable Diffusion
- 📈 Stock price prediction with LSTM
- 🏥 Medical diagnosis assistant
