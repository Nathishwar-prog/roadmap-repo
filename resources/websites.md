# 🌍 Websites & Online Courses

> The best free and paid platforms to learn technology. Learn at your own pace from the comfort of your home.

---

## 🆓 Free Learning Platforms

### General Programming
- 🔗 [freeCodeCamp](https://www.freecodecamp.org/) — Full web dev curriculum, completely free. Has a large community and certificates.
- 🔗 [The Odin Project](https://www.theodinproject.com/) — Full-stack web dev curriculum with real projects.
- 🔗 [Khan Academy](https://www.khanacademy.org/computing) — Great for absolute beginners, especially math and CS basics.
- 🔗 [CS50 (Harvard)](https://cs50.harvard.edu/) — World's most popular intro CS course, completely free.
- 🔗 [MIT OpenCourseWare](https://ocw.mit.edu/) — Free MIT course materials for CS, math, and engineering.

### Python
- 🔗 [Automate the Boring Stuff](https://automatetheboringstuff.com/) — Free online Python book.
- 🔗 [Real Python](https://realpython.com/) — High-quality tutorials (mix of free and paid).
- 🔗 [Python Docs](https://docs.python.org/3/) — Official Python documentation and tutorials.

### Web Development
- 🔗 [MDN Web Docs](https://developer.mozilla.org/) — The definitive reference for HTML, CSS, JavaScript.
- 🔗 [javascript.info](https://javascript.info/) — Modern JavaScript tutorial, completely free.
- 🔗 [CSS-Tricks](https://css-tricks.com/) — CSS guides, tips, and the Flexbox/Grid visual guides.
- 🔗 [Frontend Mentor](https://www.frontendmentor.io/) — Real design challenges to improve your HTML/CSS/JS.

### AI / Machine Learning
- 🔗 [fast.ai](https://www.fast.ai/) — Practical deep learning for coders (top-down approach, free).
- 🔗 [Google ML Crash Course](https://developers.google.com/machine-learning/crash-course) — Free ML course by Google.
- 🔗 [Hugging Face Course](https://huggingface.co/learn/nlp-course/) — NLP and Transformers, free.
- 🔗 [Kaggle Learn](https://www.kaggle.com/learn) — Short, practical ML and data science courses, free.

### SQL & Databases
- 🔗 [SQLZoo](https://sqlzoo.net/) — Interactive SQL exercises with instant feedback, perfect for beginners.
- 🔗 [SQLBolt](https://sqlbolt.com/) — Step-by-step interactive lessons covering all core SQL concepts.
- 🔗 [Mode SQL Tutorial](https://mode.com/sql-tutorial/) — Practical SQL for data analysis, free and browser-based.
- 🔗 [CS50's Introduction to Databases with SQL](https://cs50.harvard.edu/sql/) — Harvard's free SQL course.
- 🔗 [PostgreSQL Official Docs](https://www.postgresql.org/docs/) — The definitive reference for PostgreSQL.

### Algorithms & Data Structures
- 🔗 [Visualgo](https://visualgo.net/) — Visual animations of data structures and algorithms.
- 🔗 [Big-O Cheat Sheet](https://www.bigocheatsheet.com/) — Quick reference for algorithm complexity.

### DevOps & Cloud
- 🔗 [Linux Journey](https://linuxjourney.com/) — Free interactive Linux learning path, from beginner to sysadmin.
- 🔗 [Docker Official Get Started](https://docs.docker.com/get-started/) — Official Docker tutorial with hands-on labs.
- 🔗 [GitHub Actions Docs](https://docs.github.com/en/actions) — Official GitHub Actions documentation and examples.
- 🔗 [Terraform Learn](https://developer.hashicorp.com/terraform/tutorials) — Official HashiCorp Terraform tutorials (free).
- 🔗 [AWS Free Tier](https://aws.amazon.com/free/) — Hands-on AWS experience with a free tier account.
- 🔗 [Killercoda](https://killercoda.com/) — Free browser-based labs for Kubernetes, Linux, and DevOps.
- 🔗 [Play with Docker](https://labs.play-with-docker.com/) — Free browser-based Docker sandbox requiring no installation.
- 🔗 [TechWorld with Nana (YouTube)](https://www.youtube.com/@TechWorldwithNana) — Best free DevOps video content on YouTube.

### Cybersecurity
- 🔗 [TryHackMe](https://tryhackme.com/) — Beginner-friendly guided hacking rooms, free tier available.
- 🔗 [HackTheBox](https://www.hackthebox.com/) — Realistic penetration testing labs for all skill levels.
- 🔗 [PortSwigger Web Security Academy](https://portswigger.net/web-security) — Free, world-class web security labs from the makers of Burp Suite.
- 🔗 [PicoCTF](https://picoctf.org/) — CTF challenges designed by Carnegie Mellon for beginners.
- 🔗 [Cybrary](https://www.cybrary.it/) — Free cybersecurity courses for all levels.
- 🔗 [OWASP Top 10](https://owasp.org/www-project-top-ten/) — The definitive reference for web application security risks.

## 💰 Paid Platforms (Often Discounted or Free Trials)

| Platform | Best For | Cost |
|---|---|---|
| [Udemy](https://www.udemy.com/) | Comprehensive courses on everything | ~$15–20 on sale |
| [Coursera](https://www.coursera.org/) | University-level courses, certificates | Free to audit |
| [Pluralsight](https://www.pluralsight.com/) | Tech and DevOps skills | ~$29/month |
| [Frontend Masters](https://frontendmasters.com/) | Deep frontend & JS courses | ~$39/month |
| [Egghead.io](https://egghead.io/) | Short-form React/JS videos | ~$25/month |
| [Zero to Mastery](https://zerotomastery.io/) | Web dev, Python, ML courses | ~$39/month |

> 💡 **Tip:** Udemy courses go on sale frequently for ~$10–15. Never buy at full price!
