## 💻 Code Editors & IDEs

| Tool | Best For | Cost | Link |
|---|---|---|---|
| **VS Code** | All-purpose, most popular | Free | [code.visualstudio.com](https://code.visualstudio.com/) |
| **PyCharm** | Python development | Free (Community) | [jetbrains.com/pycharm](https://www.jetbrains.com/pycharm/) |
| **WebStorm** | JavaScript/TypeScript | Paid (free for students) | [jetbrains.com/webstorm](https://www.jetbrains.com/webstorm/) |
| **Neovim** | Terminal-based, highly customizable | Free | [neovim.io](https://neovim.io/) |
| **Cursor** | AI-powered code editor | Free tier | [cursor.sh](https://cursor.sh/) |

### Recommended VS Code Extensions
- 🔌 **Python** (Microsoft) — Python language support
- 🔌 **ESLint** — JavaScript linting
- 🔌 **Prettier** — Code formatter
- 🔌 **GitLens** — Supercharged Git features
- 🔌 **Thunder Client** — API testing in VS Code
- 🔌 **Docker** — Docker integration
- 🔌 **Remote SSH** — Develop on remote servers
