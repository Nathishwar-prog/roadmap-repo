# 📖 Recommended Books

> A curated collection of the best books for learning technology and software development. All recommendations are battle-tested by developers worldwide.

---

## 🐍 Python

| Book | Author | Level | Notes |
|---|---|---|---|
| *Python Crash Course* | Eric Matthes | 🟢 Beginner | Best first Python book |
| *Automate the Boring Stuff with Python* | Al Sweigart | 🟢 Beginner | Free online, very practical |
| *Fluent Python* | Luciano Ramalho | 🔴 Advanced | Deep dive into Python internals |
| *Learning Python* | Mark Lutz | 🟡 Intermediate | Comprehensive reference |
| *Python Cookbook* | David Beazley | 🔴 Advanced | Recipes for experienced developers |

---

## 🌐 Web Development

| Book | Author | Level | Notes |
|---|---|---|---|
| *HTML and CSS* | Jon Duckett | 🟢 Beginner | Visual, easy to follow |
| *JavaScript: The Good Parts* | Douglas Crockford | 🟡 Intermediate | Essential JS insights |
| *You Don't Know JS* (series) | Kyle Simpson | 🟡 Intermediate | Free on GitHub, deep JS |
| *Eloquent JavaScript* | Marijn Haverbeke | 🟡 Intermediate | Free online |
| *Node.js Design Patterns* | Mario Casciaro | 🔴 Advanced | Backend patterns |

## 🤖 AI / Machine Learning

| Book | Author | Level | Notes |
|---|---|---|---|
| *Hands-On ML with Scikit-Learn, Keras & TF* | Aurélien Géron | 🟡 Intermediate | The ML bible |
| *Deep Learning* | Goodfellow, Bengio & Courville | 🔴 Advanced | Free PDF available |
| *Python Machine Learning* | Sebastian Raschka | 🟡 Intermediate | Code-heavy, practical |
| *The Hundred-Page Machine Learning Book* | Andriy Burkov | 🟡 Intermediate | Concise and clear |
| *Natural Language Processing with Python* | Bird, Klein & Loper | 🟡 Intermediate | Free online (NLTK book) |

## 🏗️ System Design & Software Engineering

| Book | Author | Level | Notes |
|---|---|---|---|
| *Designing Data-Intensive Applications* | Martin Kleppmann | 🔴 Advanced | Must-read for backend devs |
| *System Design Interview* (Vol 1 & 2) | Alex Xu | 🟡 Intermediate | Interview focused, very practical |
| *Clean Code* | Robert C. Martin | 🟡 Intermediate | Writing readable code |
| *Clean Architecture* | Robert C. Martin | 🔴 Advanced | Software architecture principles |
| *The Pragmatic Programmer* | David Thomas & Andrew Hunt | 🟡 Intermediate | Career and craft advice |
| *Refactoring* | Martin Fowler | 🔴 Advanced | Code improvement techniques |

|---|---|---|---|
